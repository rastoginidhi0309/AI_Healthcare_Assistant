# 🏥 AI Healthcare Assistant

This project is an AI-powered Healthcare Assistant built using Streamlit. It helps users analyze their symptoms and provides basic health insights and guidance using machine learning techniques.

## 🚀 Features
- Symptom-based disease prediction
- AI-powered health suggestions
- User-friendly Streamlit interface
- Fast and interactive responses

## 🛠️ Tech Stack
- Python
- Streamlit
- Machine Learning
